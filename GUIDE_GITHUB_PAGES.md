# 🚀 Guide : Héberger votre CV sur GitHub Pages

## Votre repository GitHub
**URL :** https://github.com/labiko/cv-alpha-dsi

---

## 📋 Fichiers à uploader

Vous devez uploader **2 fichiers** dans votre repository :

1. **index.html** - Le CV (déjà créé)
2. **pf1.png** - Votre photo de profil

---

## 🔧 Étapes pour mettre en ligne votre CV

### **Étape 1 : Copier les fichiers nécessaires**

1. Copiez le fichier `pf1.png` depuis :
   ```
   C:\Users\diall\Documents\CV\pf1.png
   ```
   vers :
   ```
   C:\Users\diall\Documents\IonicProjects\Claude\RECHERCHER\DIVERS\pf1.png
   ```

### **Étape 2 : Uploader les fichiers sur GitHub**

1. Allez sur votre repository : https://github.com/labiko/cv-alpha-dsi

2. Cliquez sur **"Add file"** → **"Upload files"**

3. Uploadez ces **2 fichiers** :
   - `index.html`
   - `pf1.png`

4. Ajoutez un message de commit (par exemple : "Add CV files")

5. Cliquez sur **"Commit changes"**

### **Étape 3 : Activer GitHub Pages**

1. Dans votre repository, allez dans **"Settings"** (⚙️)

2. Dans le menu gauche, cliquez sur **"Pages"**

3. Sous **"Source"** :
   - Branch : Sélectionnez **"main"** (ou "master")
   - Folder : Sélectionnez **"/ (root)"**

4. Cliquez sur **"Save"**

5. Attendez 1-2 minutes

### **Étape 4 : Accéder à votre CV en ligne**

Votre CV sera accessible à l'adresse :

```
https://labiko.github.io/cv-alpha-dsi/
```

## 🔄 Mettre à jour votre CV

Pour modifier votre CV plus tard :

1. Modifiez le fichier `index.html` localement
2. Retournez sur GitHub → "Add file" → "Upload files"
3. Uploadez le nouveau `index.html` (il remplacera l'ancien)
4. Les modifications seront en ligne en 1-2 minutes

## ❓ En cas de problème

### La photo ne s'affiche pas ?
- Vérifiez que `pf1.png` est bien uploadé à la racine du repository
- Le nom doit être exact : `pf1.png` (pas `pf1.PNG` ou autre)

### Le CV ne s'affiche pas ?
- Attendez 2-3 minutes après activation de GitHub Pages
- Vérifiez que le fichier s'appelle bien `index.html` (pas `index.HTML`)
- Essayez d'ouvrir en navigation privée (Ctrl+Shift+N) pour vider le cache

### GitHub Pages n'est pas activé ?
- Assurez-vous d'avoir au moins 1 fichier dans le repository
- Le repository doit être **public** (pas privé)
